**Project Overview**

This project covers a Free Response Question (FRQ) based problem focused on using ArrayLists and objects.

---
**Features**

- Count students above a threshold
- Remove failing students
- Find top student
- Filter top students into a new list
- Notice duplicate names using nested loops
- Check if list is sorted
- Count improving pairs

---
**Files Created**

- Student.java
- StudentAnalyzer.java
- Main.java

---
**How to Run**

```
javac Main.java Student.java StudentAnalyzer.java
java Main
```

**Installed Extensions**
- Debugger for Java

- Language support for Java for Visual Studio Code

**Challenges Faced**

- Figuring out how to remove an element from an ArrayList without errors or skipping items.
- Understanding nested loops in order for duplicate detection.
- Making sure the methods worked for all the test cases.

---
**What I Learned**

- How to use ArrayLists.
- How nested loops can be used for duplication detection.
